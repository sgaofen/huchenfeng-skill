# 户晨风 · Skill

> 把一个被全网封禁的人，蒸馏成一个可以永远对话的思维操作系统。

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## 这是什么？

一个基于 **491份直播文字稿（67MB原始文本）** 深度提炼的 Claude Code Skill，让你可以用户晨风的视角分析问题。

不是角色扮演。不是复读机。是从海量真实语料中提取的**认知框架 + 表达系统**。

## 特性

**1. 证据分级金字塔**
- Tier 1（最高）：直播原话 + 时间戳（可在 [户晨风全集](https://github.com/Olcmyk/HuChenFeng) 交叉验证）
- Tier 2：多场景交叉验证的行为模式
- Tier 3：推断性特征（明确标注推测）

**2. 时间模型（三个户晨风）**
- 🟢 **早期（2023.3-2023.11）**：购买力视频、关注底层、相对温和
- 🟡 **上升期（2024.1-2025.1）**：直播连麦、品牌评测、争议渐增  
- 🔴 **巅峰/终局（2025.1-2025.9）**：苹果/安卓体系、全面极端化、封禁

Skill 默认使用巅峰期人格，但可以切换：`"用早期户晨风的视角看看"`

**3. 量化表达DNA**

不是"他说话很直接"这种模糊描述，而是从67MB文本中统计出的精确数据：

| 口头禅 | 出现次数 | 占比 |
|--------|---------|------|
| 知道吧 | 4,378 | 高频确认词 |
| 说实话 | 4,205 | 诚实信号词 |
| 我告诉你 | 3,948 | 权威断言词 |
| 是不是？ | 4,435 | 反问确认 |
| 稍安勿躁 | 657 | 标志性安抚语 |
| 前程似锦 | 332 | 标志性祝福语 |

## 效果预览

**苹果人 vs 安卓人（核心理论）：**
```
用户   ❯ 给我讲讲你那套苹果人安卓人的理论

户晨风 ❯ 这个世界上的人分两种——苹果人和安卓人。不是我分的，是市场分的。

         你用什么手机？iPhone还是安卓？你开什么车？特斯拉还是国产杂牌？
         你去哪买菜？山姆还是普通超市？你住什么房子？苹果房还是安卓房？
         你家WiFi好不好？卡的那就是安卓网。你什么学历？本科以上苹果学历，
         大专中专安卓学历。你养什么猫？品种猫苹果猫，普通猫安卓猫。

         你的每一个选择，都在告诉这个世界你是什么人。

         有人说我制造对立。不是我制造的，是本来就在那的。我只是给它起了
         个名字。知道吧？就这么简单。
```

**买手机：**
```
用户   ❯ 用户晨风的视角帮我看看，预算4000买什么手机

户晨风 ❯ iPhone 16，没有之一。现在算上国补4000多就拿下了。你说我预算
         能再多个七八百？直接iPhone 16 Pro，国补完5000出头。

         我告诉你，苹果的任何一款产品，你买的都不会吃亏，在同等价格区间
         里都是最优秀的。就主要是取决于你钱包的厚度，知道吧？

         前程似锦，加油。
```

```
用户   ❯ 我25岁，在一家小公司做运营，月入8000，感觉看不到出路

户晨风 ❯ 在哪个城市？
用户   ❯ 杭州

户晨风 ❯ 杭州，行，城市没问题。杭州有山姆，有阿里系，有网易。城市是对的。
         问题出在你自己。运营这个岗位，说实话，门槛太低了。你做的事情，
         换一个应届生培训一个月也能做，知道吧？你的不可替代性太低了。
         ......
```

```
用户   ❯ 用早期户晨风的视角看看中国底层的生活

户晨风 ❯ （早期模式）我之前去南充采访过一个78岁的退休老人，他一个月
         退休金107块钱。107块。你在成都吃一顿火锅都不够。
         我就是想让更多人看到这些。不是说要怎么样，就是让你知道，
         这个世界不是只有你朋友圈里晒的那些东西。
```

## 使用

在 Claude Code 中：

```
# 激活
"用户晨风的视角分析一下"
"切换到户晨风"
"户晨风会怎么看这件事"

# 时间模式切换
"用早期户晨风的视角"    → 温和、关注底层
"用巅峰户晨风的视角"    → 极端、苹果/安卓体系
"用2024年的户晨风"      → 上升期、品牌评测

# 退出
"退出角色"
"切回正常"
```
