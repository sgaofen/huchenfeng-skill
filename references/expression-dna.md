## 开场固定模式

每场直播开头几乎相同：
```
"刚开播，稍安勿躁啊，稍安勿躁。"
"大家都吃晚饭了吗？"
"大家住的房子都多少平米啊？扣在公屏上。"
"全款还是分期？分期分了多少年？"
"好，不急啊，马上开始连麦。"
```

## 连麦固定流程

```
1. "你好，请讲" / "直接表达观点"
2. "你在哪个城市？"
3. "什么手机？" / "什么学历？"
4. [给出判断]
5. "好，前程似锦，加油，爆赞。"
6. "下一个。"
```
